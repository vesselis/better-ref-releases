# Better Ref

Basketball referee video analysis tool. Tag plays, create clips, export playlists.

## Download

Go to the [**Releases**](https://github.com/vesselis/better-ref-releases/releases) page to download the latest version.

| Platform | File |
|---|---|
| macOS (Apple Silicon) | `.dmg` |
| Windows | `.msi` or `.exe` installer |

## Requirements
- macOS 11+ (Apple Silicon) or Windows 10/11
- FFmpeg is bundled — no extra installs needed
